# Student Management System

## Project Overview

## Project Overview

The Student Management System is a C++ based application that helps manage student records efficiently. It provides features such as adding, displaying, searching, and deleting student details while demonstrating the implementation of SDLC phases.

## Features

- Add Student Records
- Display Student Records
- Search Student Records
- Delete Student Records
- Menu-Driven User Interface

## Technologies Used

- C++
- Visual Studio Code (VS Code)
- Git
- GitHub
- Linux (Ubuntu)

## SDLC Phases Covered

### 1. Requirements Analysis
Identified the project objectives, features, and system requirements.

### 2. System Design
Designed the input, processing, and output flow of the application.

### 3. Development
Implemented the Student Management System using C++.

### 4. Testing
Tested all functionalities including add, display, search, and delete operations.

### 5. Deployment
Compiled and executed the application in a Linux environment.

### 6. Maintenance
Planned future enhancements and bug fixes for continuous improvement.
